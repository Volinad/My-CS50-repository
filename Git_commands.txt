Git commands from https://www.youtube.com/watch?v=71WzH4inmkg

git status          - List the files you've changed and those you still need to add or commit:

git add Git_commands.txt

git commit -m "My first git commit. Text file with git commands"            - commit to local repository

git pull            - Fetch and merge changes on the remote server to your working directory

git push            - Send changes to remote repository

-----------------------

git checkout -b new_branch_name         - create new branch (local)

git commit -am "commit comment"         - will commit all changes on tracked files, but it doesn't add new files.

touch sometext.txt          - new file for test

git add sometext.txt

git commit -m "added file"

git branch          - list of branches

git checkout master              - back to main brach

git merge new_branch
